Основной файл jafar.py: Этот файл отвечает за основной цикл программы и обрабатывает входящие команды.
Модуль tasks/weather.py: Содержит функции для работы с погодой.
Модуль tasks/voice_commands.py: Содержит функции для распознавания голосовых команд и ключевых слов.
Модуль tasks/image_generation.py: Содержит функции для генерации изображений.
Модуль tasks/general.py: Содержит общие функции, такие как преобразование текста в речь и обработка общих команд.
Файл config.py: Должен содержать ваши ключи API и другие конфигурации.
